Welcome to comic_git!

This is a website dedicated to showing off the powers of comic_git, a webcomic generator written in Python and hosted on GitHub Pages! The purpose of comic_git is to provide webcomic creators a hosting solution that is free and 100% in their control!

Click here to see a comic that goes more in-depth explaining the power and utility of comic_git!

Instructions

To learn how to use comic_git, check out the documentation.
If you'd like to jump right into creating your own webcomic, read the Getting Started section.

Note: This whole website is built by comic_git! You can see how it's built over at the GitHub repository.
